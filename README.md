# siemantik-frontend

## Prerequisites
  - [Node.js](https://nodejs.org/en/) version 8.9 or above (8.11.0+ recommended)

## Project setup
```
npm install
```

### Compile and hot-reload for development
```
npm run serve
```

### Compile and minify for production
```
npm run build
```

### Run tests
```
npm run test
```

### Run linter
```
npm run lint
```
